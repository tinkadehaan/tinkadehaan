Hoe zit deze website in elkaar?

HTML

Dit is de standaard opmaaktaal die wordt gebruikt om webpagina's te maken. Het biedt de structuur en inhoud van een webpagina en definieert wat elk element op de pagina is en hoe het eruit moet zien. Denk aan het bouwplan van een huis.

CSS

Dit wordt gebruikt om de stijl van webpagina's te bepalen, waaronder kleuren, lettertypen en ruimte. Met CSS kun je bepalen hoe elk element van een webpagina wordt weergegeven. Denk aan de verf en meubels in het huis.

Simpele website

Als je op de onderstaande knop klikt kom je op een website die goed laat zien waarmee wij aan de slag zijn gegaan. Alles op deze website is op dezelfde stukjes code opgebouwd met HTML, alleen hebben we er een handig extra programma voor gebruikt. Zie het als iemand die je huis komt inrichten.
